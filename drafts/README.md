# drafts/

Rascunhos gerados pelos agentes. **Nada aqui deve ser publicado automaticamente.**

## Princípios
- Produzir opções curtas e revisáveis.
- Sempre terminar com uma **stop condition** (ex.: 3 versões e parar).

## Estrutura
- `x/` → tweets/threads (rascunhos)
- `linkedin/` → posts LinkedIn
- `newsletter/` → edições / seções

## Regras
- Agentes geram rascunho; **Marcos aprova** antes de qualquer ação externa.
- Jarvis pode vetar (qualidade/risco), mas não deve reescrever a peça inteira.

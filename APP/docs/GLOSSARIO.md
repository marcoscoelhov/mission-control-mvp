# Glossário (PT-BR)

- **Missão**: card do board. Um objetivo com critério de sucesso.
- **PROOF**: evidência verificável de que foi feito (diff, output, screenshot, link).
- **Gate**: trava obrigatória antes de avançar (Jarvis / Marcos / Proof).
- **Risco 0 (R0)**: interno/seguro. Pode rodar sem pedir.
- **Risco 1 (R1)**: muda fluxo/contrato/arquitetura. Exige revisão do Jarvis.
- **Risco 2 (R2)**: externo (redes sociais/terceiros/dinheiro). Exige OK do Marcos.
- **Entrada (Inbox)**: onde tudo começa.
- **Delegadas (Assigned)**: missão com responsável definido.
- **Executando (In Progress)**: em execução.
- **Em revisão (Review)**: aguardando validação.
- **Proof pendente**: executou mas não provou direito.
- **Aguardando você**: depende de resposta/decisão do Marcos.

# agents/

Definições **versionadas** de cada especialista do Reino.

Regra: SOUL/AGENTS aqui são o *source of truth* humano.
O runtime do OpenClaw também possui cópias em `/root/.openclaw/agents/<id>/agent/`.

Padrão recomendado (por agente):
- `SOUL.md` (40–80 linhas)
- `AGENTS.md` (workflow + contratos de arquivos)
- `memory/` (logs diários e memória curada)
